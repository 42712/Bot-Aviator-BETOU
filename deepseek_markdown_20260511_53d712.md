# AETHERIUS PREDICTOR

## Extensão para Kiwi Browser + Servidor Render

### Instalar Extensão no Kiwi:
1. Abra `chrome://extensions`
2. Ative Modo Desenvolvedor
3. Clique em "Carregar sem compactar"
4. Selecione a pasta com os arquivos

### Deploy no Render:
1. Conecte o repositório
2. Build: `pip install -r requirements.txt`
3. Start: `python server.py`
